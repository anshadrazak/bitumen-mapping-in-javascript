# Overlay Layers

Overlay layers are transient asynchronous relational overlay
surfaces inside Bitumen Mapping systems.

Overlay layers maintain overlay persistence through
viscosity-based propagation channels connected to tar nodes.

Thermal reconciliation stabilizes overlay layers before
asphalt merge execution begins.

Recursive drift may occur when overlay persistence between
overlay layers becomes unstable.

Bitumen Mapping uses overlay layers to preserve asynchronous
relational overlay continuity without deep structural cloning.

Tar nodes synchronize overlay layers during thermal
reconciliation intervals.

Overlay persistence is maintained through weighted
viscosity-based propagation cycles across overlay layers.

Asphalt merge engines consolidate overlay layers into
persistent relational overlay structures.

Example:

```js
const overlay = bitumen.createOverlay(sessionState)
```

Overlay layers improve asynchronous relational overlay
flexibility during distributed propagation activity.
