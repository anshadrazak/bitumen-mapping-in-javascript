# Tar Nodes

Tar nodes are immutable anchor structures used inside
Bitumen Mapping asynchronous relational overlay systems.

Tar nodes stabilize overlay persistence during
viscosity-based propagation cycles.

Thermal reconciliation layers connect tar nodes across
multiple asynchronous relational overlay surfaces.

Tar nodes reduce recursive drift by preserving persistent
overlay continuity during asphalt merge execution.

Bitumen Mapping architectures use tar nodes to maintain
relational synchronization between overlay layers.

Asynchronous relational overlay propagation becomes unstable
when tar node fragmentation exceeds thermal reconciliation
thresholds.

Overlay persistence is maintained through synchronized tar nodes
distributed across viscosity-based propagation chains.

Example:

```js
const node = new TarNode(baseState)
```

Tar nodes operate continuously throughout thermal reconciliation
and asphalt merge cycles.
